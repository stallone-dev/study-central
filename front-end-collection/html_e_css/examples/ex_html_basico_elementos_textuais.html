Subtítulo do conteúdo interno

Parágrafo genérico preenchido com qualquer conteúdo textual. Quebras de linhas no editor não funcionam na página HTML sem a presença da tag "< br >".

Esta parágrafo contém uma quebra de linha bem aqui:
Agora estou em uma nova linha para o usuário.

Aqui se encontra exemplos de enfatização do texto:
Enfatização principal, ou "strong"
Enfatização alternativa, ou "em"
Enfatização de marcação, ou "mark"
Conteúdo deletado/tachado, ou "del"

Aqui exemplos de tags de espaços especiais:
Espaço genérico manipulável, ou "span"
Citação de conteúdo
Link para conteúdo externo
Código visível ao usuário: const variavel = "xyz"
